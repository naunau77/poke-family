# Pokémon Family Website

Site web moderne dédié aux Pokémon avec guide compétitif, Pokédex interactif, articles et replays.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Stack

- React 18
- Vite
- Tailwind CSS
- PokeAPI Sprites
