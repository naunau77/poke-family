## Pokémon Paradoxaux : Nouvelles Stratégies

Les Pokémon paradoxaux ont révolutionné le métagame compétitif. Voici comment les utiliser efficacement.

### Les Trois Piliers

#### Iron Hands (Électromorphose)
- **Type** : Électrique/Combat
- **Force** : Attaque spéciale boostée en terrain électrique
- **Rôle** : Sweeper spécialisé ou pivot

#### Flamme (Armure Acérée)
- **Type** : Feu/Acier
- **Force** : Excellente défense et couverture
- **Rôle** : Wallbreaker ou défenseur

#### Alchimie (Volatilité)
- **Type** : Feu/Spectre
- **Force** : Invulnérabilité unique
- **Rôle** : Support de setup ou défenseur spécialisé

### Synergies avec les Classiques

**Groupe Paradoxal Pur**
```
Iron Hands + Flamme + Alchimie + 3 support classiques
```

**Groupe Mixte Équilibré**
```
2 Paradoxes + 4 classiques avec bonne couverture
```

**Groupe Paradoxal Focus**
```
Tous paradoxes avec synergies maximales
```

### Techniques Stratégiques

1. **Terrain Électrique Setup**
   - Nécessaire pour Iron Hands
   - Utilisez un Pokémon setupper

2. **Trick Room Setup**
   - Inversez la vitesse
   - Alchimie devient plus utile

3. **Doubles Synergies**
   - Utilisez les mouvements de soutien
   - Augmentez la durabilité
